### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/suhailkakar/React-Native-Video-Chat
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or 
   
     ```sh
   yarn install
   ```
3. Run the app
   ```sh
   react-native run-ios
   ```
   or 
   
     ```sh
   react-native run-android
   ```
